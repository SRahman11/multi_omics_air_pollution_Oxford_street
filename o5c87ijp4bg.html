Sensitivity analysis

To assess the sensitivity of our analysis to outlier data points in the OMICs features, the analysis was run multiple times with caps on the OMICs features to remove outliers above or below specified percentiles (specifically, 5%, 10% and 20%).

Multi-OMIC integrative analysis

The GLS model identified a set of biomarkers for each of the OMIC levels that have significant associations with the experimentally-induced change in TRAP exposure. Based on the assumption that individuals share common biological responses to TRAP pollutants, resulting in similar changes in OMICs measurements, integrative analysis then aims to discover statistical associations that link the changes in one OMIC level to another. For example, the differential expression of a transcript mediating the change in measured levels of a metabolite. The integrative techniques applied were network analysis and sparse partial least squares (sPLS) regression. Network analysis takes a broad perspective and isolates statistically significant linkages within and between OMICs levels, while sPLS focuses specifically on between-OMIC associations. 

Network Analysis

To investigate and visualise biological pathways and relationships within and between OMICs levels, a range of network analysis techniques were applied. At the most basic level, marginal networks examined the pairwise relationships between biomarkers. P-values for covariance of each biomarker pairing were derived, and corrected for multiple testing using a Bonferroni adjustment. As four levels of OMIC were being analysed, the correlation matrix contained 10 inter-OMIC and intra-OMIC regions, and each region was adjusted for false discovery according to the size of the region (and therefore the number of tests conducted in that region).
